# AI-Powered-Professional-Email-Drafting-Assistant-1-
AI-powered email assistant that automatically drafts professional and polite replies to incoming Gmail messages using a human-in-the-loop workflow. Built with Relay and LLMs to improve productivity while keeping users in full control.

An AI-powered email drafting assistant designed to improve productivity and communication quality.

When a new email is received in Gmail, the system automatically analyzes the message and generates a short, professional, and polite reply as a draft. The workflow intentionally stops at draft creation, ensuring full human oversight before sending.

This project demonstrates event-driven AI automation, prompt engineering, and human-in-the-loop system design using Relay.app, Gmail, and large language models. It is suitable for personal, professional, and enterprise email workflows.
